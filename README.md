## Prerequisites
1. Clang/LLVM 20 (default environment setting)
    ```
    $ wget https://github.com/llvm/llvm-project/releases/download/llvmorg-20.1.8/llvm-project-20.1.8.src.tar.xz
    $ cmake -GNinja -S llvm-project-20.1.8.src/llvm -B llvm_build -DCMAKE_INSTALL_PREFIX=`pwd`/llvm_20.1.8_install -DCMAKE_BUILD_TYPE=Release -DLLVM_TARGETS_TO_BUILD=X86 -DBUILD_SHARED_LIBS=OFF -DLLVM_ENABLE_PROJECTS="clang;compiler-rt;clang-tools-extra;lld"
    $ cmake --build llvm_build --target install
    ```

2. Clang/LLVM 12
    ```
    $ gaster23.swtv:/ssd_home/yangheechan/downloads/llvm-project-12.0.1.src.editted.tar.gz
    $ cmake -GNinja -S llvm-project-12.0.1.src/llvm -B llvm-12.0.1-build -DCMAKE_INSTALL_PREFIX=`pwd`/llvm_12.0.1_install -DCMAKE_BUILD_TYPE=Release -DLLVM_TARGETS_TO_BUILD=X86 -DBUILD_SHARED_LIBS=OFF -DLLVM_ENABLE_PROJECTS="clang;compiler-rt;clang-tools-extra;lld" CC=gcc-9 CXX=g++-9 -DCOMPILER_RT_HAS_FLOAT16=OFF
    $ cmake --build llvm-12.0.1-build --target install
    ```

## Procedure
[important] ``<OUTPUT_DIR>`` must end with the name of ``<TARGET_PROGRAM>``.

1. Target program compilation
    ```
    time python3 fuzzer-xyz/src/scripts/compile.py -o <OUTPUT_DIR> -t <TARGET_PROGRAM> -f <FUZZER>
    ```

2. Fuzzing
    * Fuzzing on local machine
        ```
        timeout --signal=2 24h python3 fuzzer-xyz/src/scripts/fuzz.py -o <OUTPUT_DIR> -t <TARGET_PROGRAM> -f <FUZZER> -e <EXPERIMENT_NAME> [-p <NUM_FUZZERS>] [-fid <FUZZ_ID>]
        ```
        * If ``-fid`` is not given, ``<FUZZ_ID>`` will be set to ``MMDD_HHMMSS`` form.
        * Default of ``<NUM_FUZZERS>`` is 4 processes.
    * Fuzzing on server machine
        ```
        time python3 fuzzer-xyz/src/scripts/fuzz_on_server.py -o <OUTPUT_DIR> -t <TARGET_PROGRAM> -f <FUZZER> -s <SERVER_ADDRESS> -e <EXPERIMENT_NAME> [-p <NUM_FUZZERS>] [-d <DURATION>]
        ```
        * Default of ``<DURATION>`` is 24h.

3. Execute fuzz generated inputs for gcov/bbcov coverage
    * Execution on local machine
        ```
        time python3 fuzzer-xyz/src/scripts/get_cov.py -o <OUTPUT_DIR> -t <TARGET_PROGRAM> -f <FUZZER> -e <EXPERIMENT_NAME> -fid <FUZZ_ID> [-w <WINDOW_SIZE>] [--no-cleaup]
        ```
        * ``<WINDOW_SIZE>`` is the size in which inputs generated from fuzzers are grouped.
        * Default of ``<WINDOW_SIZE>`` is 10 minutes.
    * Execution on server machines
        ```
        time python3 fuzzer-xyz/src/scripts/get_cov_on_server.py -o <OUTPUT_DIR> -f <FUZZER> -e <EXPERIMENT_NAME> [-p <PARALLEL_TASKS>] [-w WINDOW_SIZE]
        ```
        * ``<PARALLEL_TASKS>`` decides the number of parallel execution in a single server.
        * Default of ``<PARALLEL_TASKS>`` is 8 taks.

4. Make coverage graph
    ```
    time python3 fuzzer-xyz/src/scripts/graph_cov.py -o <OUTPUTS_DIR> -e <EXPERIMENT_NAME>
    ```
    * This script produces a line graph of the fuzzers within ``./data/coverage/`` directory.
